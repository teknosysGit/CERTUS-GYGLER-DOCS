# Release Notes

## February Release

### New Features
- Added support for new hardware components.
- Improved user interface for better navigation.

### Bug Fixes
- Fixed an issue where the device would not connect to certain networks.
- Resolved a bug that caused unexpected shutdowns during operation.

### Performance Improvements
- Optimized software for faster response times.
- Reduced memory usage for better performance on older devices.

### Known Issues
- Some users may experience connectivity issues with specific network configurations. We are actively working on a fix
- The device may not function properly in extreme temperatures. Please refer to the specifications for operating conditions.
