## WEB CONTROL
CERTUS WEB CONTROL is a chargeable add-on that enables maintenance functions to be carried out on the {{ variables.product.name.en }} via a web browser on a smartphone or tablet.

## CERTUS CONTROL
{{ variables.product.software.name.en }} is the software for controlling the {{ variables.product.name.en }}. It is included in the delivery of the dispenser and can be installed on a Windows computer. The software allows you to set up and operate the dispenser, create and manage dispensing protocols, and monitor the dispensing process.
